# Scatterplot

x <- c(0.5, 2, 4, 8, 12, 16) # Define the x values
y1 <- c(1, 1.3, 1.9, 3.4, 3.9, 4.8) # Define the y1 values
y2 <- c(4, .8, .5, .45, .4, .3) # Define the y2 values

par(las=1, mar=c(4, 4, 2, 4), cex=.7) # las=1: Make axis labels always horizontal
# mar=c(4, 4, 2, 4): Set margins (bottom, left, top, right)
# cex=0.7: Set the text size to 70% of the default size

plot.new() # Create a new blank plot
plot.window(range(x), c(0, 6)) # Set up the plotting window with defined x and y ranges
# range(x): Set the range of x-axis based on x values
# c(0, 6): Set the y-axis range from 0 to 6

lines(x, y1) # Draw a line connecting the points defined by x and y1 values
lines(x, y2) # Draw a line connecting the points defined by x and y2 values

points(x, y1, pch=16, cex=2) # Plot points for the first set of data (x, y1) using solid circles (pch=16)
# cex=2: Set the point size to twice the default size
points(x, y2, pch=21, bg="white", cex=2) # Plot points for the second set of data (x, y2) using open circles with a white fill (pch=21, bg="white")
# cex=2: Set the point size to twice the default size

par(col="gray50", fg="gray50", col.axis="gray50") # Set the color for lines, text, and axes to gray50

axis(1, at=seq(0, 16, 4)) # Draw the x-axis with ticks at positions 0, 4, 8, 12, and 16
# The first number in the sequence (0) stands for the starting point of the axis ticks
axis(2, at=seq(0, 6, 2)) # Draw the left y-axis with ticks at positions 0, 2, 4, and 6
axis(4, at=seq(0, 6, 2)) # Draw the right y-axis with ticks at positions 0, 2, 4, and 6

box(bty="u") # Draw a box around the plot with only the top and bottom borders

mtext("Travel Time (s)", side=1, line=2, cex=0.8) # Add x-axis label: "Travel Time (s)" at the bottom (side=1)
# line=2: Position the label two lines away from the axis
# cex=0.8: Set the text size to 80% of the default
mtext("Responses per Travel", side=2, line=2, las=0, cex=0.8) # Add left y-axis label: "Responses per Travel" (side=2)
# line=2: Position the label two lines away from the axis
# las=0: Make the text parallel to the axis
# cex=0.8: Set the text size to 80% of the default
mtext("Responses per Second", side=4, line=2, las=0, cex=0.8) # Add right y-axis label: "Responses per Second" (side=4)
# line=2: Position the label two lines away from the axis
# las=0: Make the text parallel to the axis
# cex=0.8: Set the text size to 80% of the default

par(mar=c(5.1, 4.1, 4.1, 2.1), col="black", fg="black", col.axis="black") # Restore the default graphical parameters
# mar=c(5.1, 4.1, 4.1, 2.1): Set margins back to default values
# col="black", fg="black", col.axis="black": Reset colors to black

The image depicts Anscombe’s quartet, a set of four different datasets that are constructed to have nearly identical simple descriptive statistics (e.g., mean, variance, correlation, and linear regression line) but very different distributions and patterns.

Comparison of the Regression Models:

Top Left (Dataset 1) Top Right (Dataset 2) Bottom Left (Dataset 3) Bottom Right (Dataset 4)
Features of data The data points form a roughly linear relationship. The data points form a nonlinear pattern, resembling a curve. There is a strong linear relationship, but one point is an outlier. All the points, except one, are nearly identical, with one extreme outlier on the x-axis.
The relationship between data and regression line The data points fit well with the regression line. Though computed to fit linearly, the regression line does not capture this non-linear relationship well. This single outlier’s presence significantly affects the regression line, shifting it away from the expected fit. The regression line suggests a relationship, but the x-values’ lack of variation means the model is not reliable.
The degree of agreement between the data and the regression line This dataset behaves as expected, showing a good linear fit with minimal deviation. This illustrates that linear regression may not always be the best fit for curved data. This demonstrates the sensitivity of linear regression to outliers. This illustrates how a single influential point can distort a regression analysis.

Even though all four datasets have similar linear regression results (same slope, intercept, and R-squared values), their visual patterns are significantly different. This highlights the importance of visualizing data before relying on statistical models, as summary statistics alone can be misleading.

# Scatterplot with refined elements
x <- c(0.5, 2, 4, 8, 12, 16)
y1 <- c(1, 1.3, 1.9, 3.4, 3.9, 4.8)
y2 <- c(4, .8, .5, .45, .4, .3)

# Set up graphical parameters
par(family="serif", las=1, mar=c(4, 4, 2, 4), cex=0.7) 

# Start a new plot
plot.new()
plot.window(range(x), c(0, 6))

# Draw lines for y1 and y2 with custom colors
lines(x, y1, col="blue", lwd=2)
lines(x, y2, col="orangered3", lwd=2)

# Use custom plotting characters and colors for points
points(x, y1, pch=17, cex=2, col="darkblue")   # pch=17: filled triangle
points(x, y2, pch=15, cex=2, col="orange")     # pch=15: filled square

# Customize axis colors and text
par(col="gray50", fg="gray50", col.axis="gray50")
axis(1, at=seq(0, 16, 4), col.axis="forestgreen", lwd=2)  # Custom x-axis color
axis(2, at=seq(0, 6, 2), col.axis="steelblue", lwd=2)   # Custom left y-axis color
axis(4, at=seq(0, 6, 2), col.axis="darkorange", lwd=2)  # Custom right y-axis color

# Add a customized box with specific line width
box(bty="u", lwd=2)

# Add axis labels with custom color and size
mtext("Travel Time (s)", side=1, line=2, cex=0.8, col="darkgreen")

4. How about with ggplot2? (use tidyverse package)

# Load ggplot2
library(ggplot2)

# Prepare data frame
data <- data.frame(
  x = c(0.5, 2, 4, 8, 12, 16),
  y1 = c(1, 1.3, 1.9, 3.4, 3.9, 4.8),
  y2 = c(4, 0.8, 0.5, 0.45, 0.4, 0.3)
)

# Create charts using ggplot
ggplot(data, aes(x = x)) +
  
  # Draw y1 polyline and points
  geom_line(aes(y = y1), color = "blue", linewidth = 1) +  # 使用 linewidth 替換 size
  geom_point(aes(y = y1), shape = 17, size = 3, color = "darkblue") +
  
  # Draw y2 polyline and points
  geom_line(aes(y = y2), color = "orangered3", linewidth = 1) +  # 使用 linewidth 替換 size
  geom_point(aes(y = y2), shape = 15, size = 3, color = "orange") +
  
  # Customize x- and y-axis labels and colors
  scale_x_continuous(name = "Travel Time (s)", breaks = seq(0, 16, 4), limits = c(0, 16), 
                     labels = seq(0, 16, 4), expand = c(0, 0)) +
  scale_y_continuous(
    name = "Responses per Travel",
    sec.axis = sec_axis(~ ., name = "Responses per Second", breaks = seq(0, 6, 2))
  ) +
  
  # Annotate text
  annotate("text", x = 4, y = 5, label = "Bird 131", color = "deeppink2", family = "serif", fontface = "italic", size = 6) +
  
  # Theme customization, adjust axis and text color
  theme_minimal(base_family = "serif") +
  theme(
    axis.title.x = element_text(size = 12, color = "darkgreen"),
    axis.title.y = element_text(size = 12, color = "navy"),
    axis.text.x = element_text(color = "forestgreen"),
    axis.text.y = element_text(color = "steelblue"),
    axis.text.y.right = element_text(color = "orangered2"),
    axis.title.y.right = element_text(size = 12, color = "orangered2"),
    panel.border = element_blank(),
    axis.line = element_line(color = "gray50", linewidth = 1)  # 使用 linewidth 替換 size
  )
